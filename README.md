# SmartBuy Supermarket Billing System

## Project Description
A Python-based billing system for SmartBuy Supermarket to replace manual calculations. Solves incorrect totals, slow checkout, and messy handwritten receipts by automating the billing process.

## How the Program Works
1. Cashier enters product name, quantity, and price per unit
2. System calculates subtotal for each item using lists
3. Applies 10% discount if total exceeds Le 500
4. Generates formatted receipt with all items and final total
5. Allows processing multiple customers using loops until cashier exits

## How to Run the Program
1. Ensure Python 3 is installed
2. Download `Smartbuy_billing.py`
3. Open terminal/CMD in the file location
4. Run: `python Smartbuy_billing.py`
5. Follow prompts to enter customer purchases

## Features
- Multiple product entry per customer
- Automatic total calculation
- 10% discount for purchases over Le 500
- Multiple customer processing
- Clean formatted receipt output
- Error handling for invalid inputs
